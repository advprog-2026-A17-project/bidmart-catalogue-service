# Software Deployment

## CI

[`.github/workflows/ci.yml`](../.github/workflows/ci.yml):

- `./gradlew test jacocoTestReport`
- Artifact: `catalogue-service-test-coverage`
- Sonar job when `SONAR_TOKEN` present: `./gradlew sonar`

## Docker

- [`Dockerfile`](../Dockerfile)
- Compose: `catalogue-service` exposes `8081`, `9091`
- Depends on: `catalogue-db`, `rabbitmq`

## Metrics

- Job `bidmart-catalogue-service` in Prometheus
- Grafana dashboards via infrastructure provisioning

## Environment

From `bidmart-infrastructure/docker-compose.yml`:

- `SPRING_DATASOURCE_URL` → `catalogue-db`
- `SPRING_RABBITMQ_*`, `BIDMART_EVENTS_EXCHANGE`
