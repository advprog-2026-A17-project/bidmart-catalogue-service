- DTOs: `dto/ListingSummaryResponse.java` separate from JPA `Listing` entity.

## Secure coding

- **Auth interceptor** rejects unauthenticated seller mutations.
- **CatalogAccessPolicy** prevents cross-seller listing access.
- Listing ownership enforced on update/delete paths in `ListingServiceImpl`.
- No trust of client-supplied seller id without gateway/JWT identity.
