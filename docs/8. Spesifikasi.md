# Spesifikasi — Modul 2 (Katalog dan Manajemen Listing)

## Tanggung jawab

CRUD listing, pencarian, kategori, lifecycle listing lelang, dan sinkronisasi state dari modul lelang.

## Fitur

- CRUD listing + pencarian (keyword, kategori, harga, status, `endingWithinHours`)
- Summary endpoint untuk validasi lelang
- Kategori hierarkis (tree)
- Lifecycle: publish, deactivate, extend, close, won, unsold, cancel
- Pembatasan update saat `hasBids`
- Enforce seller via header gateway (`X-User-Id`)
- Konsumen RabbitMQ: `auction.bid-placed.v1`, `auction.ended.v1`
- gRPC summary untuk layanan internal

## Integrasi

| Modul | Arah |
|-------|------|
| 3 Lelang | HTTP validasi + event harga/status |
| 7 Gateway | Autentikasi seller/admin |
